## 👍🏻 Como baixar

```bash

    // Clonar repositório
    $ git clone https://github.com/Mkroning/PointControl-Backend

    // Acessar diretório
    $ cd PointControl-Backend

    // Instalar dependências
    $ yarn install

    // Iniciar projeto
    $ yarn start
